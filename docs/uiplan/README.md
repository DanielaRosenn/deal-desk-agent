# UiPlan Documentation

UiPlan is the planning contract for this repository. It produces a three-file
bundle:

- `spec.md` - scope, requirements, architecture intent
- `plan.md` - implementation design and routing
- `tasks.md` - executable, verifiable task list

## Start Here

1. Read `docs/uiplan/HOW_TO_USE.md`.
2. Run UiPlan generation (`full` or staged `constitution -> ground -> spec -> clarify -> plan -> tasks`).
3. Run UiPlan review and resolve findings.
4. Accept and publish when ready.
