# Cloned Project Setup

After cloning this repository:

1. Install `@uipath/cli` globally:
   - `npm install -g @uipath/cli`
2. Create and activate a Python environment.
3. Install root Python dependencies.
4. Install `studio/api` dependencies and run the API.
5. Install `studio/web` dependencies and run the web UI.

Use `README.md` for exact command blocks.
